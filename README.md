# flix-pretty

A pretty print library for Flix.

License: Apache 2.0

Note - v0.11.0 changes the module names / namespace to `FlixPretty` 
the top level module and `FlixPretty.PrettyClass` the module 
containing the `Pretty` class.

## Acknowledgments

The pretty printer a Flix implementation of Christian Lindig's code 
from the paper "Strictly Pretty".

https://lindig.github.io/papers/strictly-pretty-2000.pdf

The `Pretty` class is a Flix implementation of Haskell's 
`Text.PrettyPrint.HughesPJClass`.
